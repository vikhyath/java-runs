java-runs
=========

exercises in java

* HashMapFileParser: Reads a file input-1.txt like:

A B C D
a alpha first 1
b beta second 2
c gamma third 3
d delta fourth 4

and converts it into a key value pair:
{d={D=4, B=delta, C=fourth}, b={D=2, B=beta, C=second}, c={D=3, B=gamma, C=third}, a={D=1, B=alpha, C=first}}
