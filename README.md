java-runs
=========

exercises in java

* HashMapFileParser: Reads a file input-1.txt like:<br>

  A B C D<br>
  a alpha first 1<br>
  b beta second 2<br>
  c gamma third 3<br>
  d delta fourth 4<br>
  
  and converts it into a key value pair:<br>
  {d={D=4, B=delta, C=fourth}, b={D=2, B=beta, C=second}, c={D=3, B=gamma, C=third}, a={D=1, B=alpha, C=first}}
